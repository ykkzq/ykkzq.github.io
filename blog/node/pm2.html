PM2 是 node 进程管理工具，可以利用它来简化很多 node 应用管理的繁琐任务，如性能监控、自动重启、负载均衡等，而且使用非常简单。 全局安装，简直不能更简单。

npm install -g pm2

用 express 应用来举例。一般我们都是通过 npm start 启动应用，其实就是调用 node ./bin/www。那么，换成 pm2 就是

pm2 start ./bin/www --watch(--watch：监听应用目录的变化，一旦发生变化，自动重启。如果要精确监听、不监听的目录，最好通过配置文件。)

重启

pm2 restart ./bin/www

停止

停止特定的应用。可以先通过 pm2 list 获取应用的名字（--name 指定的）或者进程 id。

pm2 stop app_name|app_id

如果要停止所有应用，可以

pm2 stop all

删除

类似 pm2 stop，如下

pm2 delete app_name|app_id

pm2 delete all

查看进程状态

pm2 list
